create - 创建订单

POST /product/pevc/create

  • 请求(Request)
  • 响应(Response)
  • 例子(Example)

Param:

参数名称 参数类型 是否必须 参数说明
id Number 是 id should match {"gt":2,"lt":20}

Body:

参数名称 参数类型 是否必须 参数说明
name String 是 name should match {"like":""} and name.length should match {"gt":2,"lt":20};
price String 是 price.length should match {"gt":2,"lt":20};
date Number 否 date should match {"neq":10}
length Number 否 length should match {"eq":2}
height Number 否 height should match {"gte":0,"lte":100}
weight Array 否 weight should match {"eq":[100,98]} and weight.length should match {"gte":2,"lte":20};
